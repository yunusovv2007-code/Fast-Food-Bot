# EVOS Fast Food Ordering Bot

A professional Telegram bot designed for automating food ordering processes. This project follows a modular structure and implements a complete user registration and menu management system.

## Features
- **User Registration:** Multilingual (Uzbek/Russian) registration flow.
- **Dynamic Menu:** Categorized product listings fetched from SQLite database.
- **Modular Design:** Separate files for database handling, handlers, and global constants.
- **State Management:** Uses `user_data` to track user progress within the bot.

## Tech Stack
- **Language:** Python 3.10+
- **Library:** `python-telegram-bot` (v13.15)
- **Database:** SQLite3
